## QR Code Encoder and Decoder

A simple Python project that demonstrates how to generate QR codes from any data and decode QR codes back into readable text using popular Python libraries.

- Generate QR codes from any text or data

- Save generated QR codes as image files (.png)

- Decode QR codes from image files

**Python 3 Libraries used.**

`qrcode` – for QR code generation

`Pillow` (PIL) – image handling

`pyzbar` – QR code decoding

`typing` – for type hints
